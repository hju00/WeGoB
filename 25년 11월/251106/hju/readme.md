# [pro] 전기차여행

## 문제 분석
정점과 간선이 존재하고 간선에 비용이 존재 -> 다익스트라

## 조건
1. 전염병이 존재하고 발생 시기가 존재
2. 전염병은 도시에 연결되어 있는 도로의 시간이 지나면, 다른 도시로 전파됨
3. 전염병이 있는 도시로는 갈 수 없음
4. 도로를 이동할 때 배터리를 소모
5. 각 도시에는 배터리를 1시간마다 충전할 수 있음
6. 충전하고 있을 때 전염병이 올 경우 실패


## 시간복잡도
1. init() X 1
2. add() X 3,000
3. remove() X 900
4. cost() X 200

## 전략
- 자동차가 이동 하는 다익스트라를 어떻게 구현하지?
기존의 다익스트라는 dist 배열을 1차원으로 구현해서 최소 비용만 계산함
- 배터리를 충전하는 시간도 고려해야하므로 배터리까지 고려하는 2차원 dist 배열을 생각
dist[도시][배터리 잔량] 로 dp + dijkstra
- 전염병을 구현하는 방법??
- 각 도시에 최초로 전염병이 발생하는 시간을 계산해서 자동차의 다익스트라에 적용하자
- 각 도시마다 전염병이 발생하는 시간?? -> 이것도 다익스트라 쓰자

- cost() 에 다익스트라가 2번 실행되니까 cost() 의 시간복잡도 = (E logV) X 2
- (7,000 X log500) X 2 = 약 37,000
- 37,000 X 200 = 7,400,000
