## 문제 설명
- 각 역은 1 ~ N까지 ID값을 가진다.
- 최소 환승 횟수를 구해야한다.
- init에선 각 열차(노선)의 시작, 종착, 간격을 주고 있다.
  - 정차 역의 간격은 3 <= mInterval <= 50  
  - 시작역은 N-3을 넘을 수 없다. 
  - 종착역은 시작역보다 크고 N보다 같거나 작음.
- add에선 열차(노선)을 추가하고 있다. 
- remove에선 mId로 열차를 제거한다.
- calculate에선 시작역에서 종점역으로 가는데 걸리는 최소 환승 횟수를 구한다.
  - 최소 환승횟수: 0 ~ 최대 환승 OR -1(이동 불가)
  - sId와 eId가 같은 경우는 없다.

## 문제 분석
- add <= 40,000
- remove <= 60,000
- 🌟calculate <= 120,000
