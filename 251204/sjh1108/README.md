# Idea
## 환승 횟수를 구해야 한다
환승 횟수 최소를 어떻게 구하는지가 핵심 로직

## 기차를 기준으로 환승해서 갈 수 있는 위치 계산
1. 기차를 기준으로 환승 할 수 있는 기차를 미리 계산해두자
2. 그러면 삭제는 어떻게 하지? -> LazyFetch 적용해두자

## 환승 횟수 계산
1. 다익인가? 근데 가중치가 매번 동일하네? 그냥 BFS구나
2. 출발지에서 탑승 가능한 열차 모두 Queue에 삽입
3. 해당 열차에서 환승할 수 있는 기차는 이미 계산되어서 List형태로 저장되어 있기에 활용해서 Queue에 삽입

# Tips
## 객체 선 생성 (Memory Pool)
1. 자바에서 객체 생성은 비용이 많이 드는 작업
2. 미리 생성해서 내부 밸류만 수정하자
    * 왜 미리 생성하는 행위가 비용이 적게 들까? 결국 생성은 똑같이 하지 않나?
    * 이 문제에서는 'UserSolution'을 한번 생성하고 계속 사용하는 로직으로 작성되어 있기 때문
    -> 한번 생성한 객체가 계속 사용될 수 있다는 의미
