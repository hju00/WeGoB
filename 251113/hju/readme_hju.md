# [pro] 삼국지게임 문제 풀이

## 문제 분석

크게 3가지 기능 존재
1. 동맹
- 전투 발생 시 동맹 영토 모두 함께 공격 및 방어
- 동맹 체결 시 서로의 모든 동맹까지 같이 동맹
- 동맹 체결 시 동맹 영토 중 적대 관계가 존재 시 동맹 불가
- 동맹 체결 시 서로의 적대 영토도 같이 병합

2. 전투
- 동맹 영토끼리는 전투 불가
- 전투하려는 영토의 근처에 동맹 영토가 존재해야 전투가 가능
- 전투 발생 시 공격 영토의 동맹과 방어 영토의 동맹은 전부 적대

3. 모집
- 주어진 군주의 영토에 mNum의 병사 추가
- 주어진 군주를 포함한 모든 동맹 영토에 mNum의 병사 추가

## 알고리즘 선정
동맹과 적대 관계를 유니온-파인드를 사용해서 구현하는 문제구나

## 시간 복잡도
- init() X 1
- ally() X 8,000
- attack() X 8,000
- recruit() X 13,000

## 자료 구조
    HashMap<String, Integer> monarchDB
    int[N][N] soldier
    int[N * N][2] parent
    HashSet<Integer>[N * N] group
